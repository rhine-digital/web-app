# Rhine Web-App
This is build on React + GraphQL + MongoDB for testing purposes.

# Hightlight
- GrahpQl
- React
- NodeJs
- React-bootstrap(Working in Progress)

Install all dependencies
```sh
npm install
```

Run the server
```sh
npm start server
```

Run the frontend
```sh
npm start client
```

Run the full-stack
```sh
npm start dev
```
